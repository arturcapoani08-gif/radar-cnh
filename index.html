✅ Configurado — botões de Subir/Baixar ativos.
Configure o token GitHub para habilitar a sincronização.
Como funciona:
Você controla manualmente o que acontece — Subir envia seus dados para a nuvem, Baixar traz os dados da nuvem para este celular. Não há sync automático.
1
Acesse github.com e crie conta gratuita
2
Vá em github.com/settings/tokens → Generate new token (classic)
3
Marque apenas o escopo gist → Generate token
4
Cole o token em Config e salve
5
No celular A: clique ⬆️ Subir — o ID do Gist será criado automaticamente
6
No celular B: mesmo token + mesmo ID do Gist → clique ⬇️ Baixar
7
O ID do Gist fica salvo na aba Config. Copie para todos os celulares.
